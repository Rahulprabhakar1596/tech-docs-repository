get /weather
Retrieve current weather data for a specified city. (weatherGet)

Query parameters

Bengaluru (required)
Query Parameter — City name (e.g., Bengaluru) or city name, state, and country code (e.g., London,uk).
appid (required)
Query Parameter — c181b573f61c543060dd0715dd05e979
metric (optional)
Query Parameter — Units of measurement (standard, metric, imperial). Default is standard. default: standard
lang (optional)
Query Parameter — Language code (e.g., en, es). Default is en. default: en

Return type

inline_response_200

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.

Responses

200

Successful response with current weather data. inline_response_200

401

Unauthorized - Invalid or missing API key. inline_response_401

404

Not Found - City not found or incorrect endpoint. inline_response_404
